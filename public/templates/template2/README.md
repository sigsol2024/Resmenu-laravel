# Template 2: Salt and Social Design

## Overview

A modern, contemporary restaurant menu template featuring:
- Tailwind CSS-based design
- Hero section with restaurant branding
- Featured menu items showcase
- Full category-based menu display
- Responsive mobile-first design
- Dark mode support

## Features

- **Hero Section**: Large banner with restaurant name, description, and call-to-action buttons
- **Featured Items**: Displays first 6 menu items prominently
- **Category Sections**: Full menu organized by categories
- **Social Integration**: Links to Instagram, Facebook, Twitter
- **Contact Information**: Phone, email, and address display
- **WhatsApp Integration**: Order online button links to WhatsApp

## Design Elements

- Primary Color: #ea2a33 (Red)
- Fonts: Be Vietnam Pro, Noto Sans
- Layout: Max-width 1440px container
- Responsive breakpoints: Mobile, Tablet, Desktop

## Customization

This template uses Tailwind CSS via CDN and Material Symbols for icons. The design adapts to:
- Restaurant name and description
- Logo and hero images
- Menu items and categories
- Contact information
- Social media links
