# Template 1: Modern Classic

## Description

Clean, modern design with rounded corners and elegant typography. Based on the original Framer design, converted to a dynamic PHP template.

## Features

- Hero section with restaurant logo and name
- Responsive category images with rounded corners
- Card-based menu item layout
- Search and filter functionality
- Integrated footer with map, social media, and contact info
- Full customization support via CSS variables

## Design Elements

- **Hero Section**: Centered logo, title, and description with "Place an order" button
- **Category Images**: Large rounded images (463px border-radius) with perspective transform
- **Menu Items**: Card layout with rounded corners, images, and pricing
- **Footer**: Light background (#ffeacf) with map, address, and social links

## Responsive Breakpoints

- Desktop: 1200px+
- Tablet: 810px - 1199px
- Mobile: < 810px

## Customization Variables

All customization settings are applied via CSS variables:

- Menu title: color, size, font
- Price: color, size, font
- Description: color, size, font
- Category title: color, size, font
- Background colors: page, header
- Primary/secondary colors: buttons and accents

## File Structure

- `index.php` - Main template file
- `style.css` - Template-specific styles

## Usage Notes

- Category images should be at least 736x736px for best results
- Menu item images work best at 300x300px or larger
- Logo should be square (200x200px recommended)
- WhatsApp link is required for "Place an order" buttons to appear
